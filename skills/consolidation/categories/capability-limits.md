# 能力边界问题

## 定义

AI 模型的固有局限导致的问题，无法通过改进文档或 Skill 解决。包括：上下文窗口限制、模型差异、推理能力边界。

## 信号识别

| 信号来源 | 字段 | 含义 |
|---------|------|------|
| ooda-coder | execution-log.md 卡点 | AI 因上下文窗口不足而丢失信息 |
| ooda-act | 自我纠正记录 | AI 因推理能力不足而无法修复编译错误 |
| ooda-eval | 待确认事项 | AI 因上下文不足而无法判断测试失败原因 |
| debugger | diagnosis-report.md 根因 | AI 因推理能力不足而无法定位根因 |
| reviewer | 审查维度结果 | AI 因上下文不足而遗漏审查维度 |
| architecture-guard | 违规项详情 | AI 因上下文不足而遗漏架构违规 |

## 判断逻辑

**上下文窗口限制**：
- if AI 在处理大量文件时丢失信息 → 上下文窗口限制
- if AI 在长对话中忘记之前的输出 → 上下文窗口限制
- if AI 因上下文压缩而产生幻觉 → 上下文窗口限制

**模型差异**：
- if 同一 Skill 在不同模型上表现差异很大 → 模型差异
- if 弱模型无法完成强模型能完成的任务 → 模型差异

**推理能力边界**：
- if AI 无法理解复杂的业务逻辑 → 推理能力边界
- if AI 无法进行多步推理 → 推理能力边界
- if AI 无法处理模糊的边界情况 → 推理能力边界

**区分其他类别**：
- if 问题出在文档不完整（文档问题，不是能力问题）→ 文档质量
- if 问题出在 Skill 设计不合理（设计问题，不是能力问题）→ Skill 设计
- if 问题出在用户输入不清晰（输入问题，不是能力问题）→ 用户输入
- if 问题出在 AI 没有遵循约束（行为问题，不是能力问题）→ 执行偏差

## 映射目标

| 问题类型 | 映射目标 | 演化操作 |
|---------|---------|---------|
| 上下文窗口限制 | 框架设计 | 优化信息密度、分批处理、子代理隔离 |
| 模型差异 | 框架设计 | 适配不同模型的能力等级 |
| 推理能力边界 | 框架设计 | 简化任务、提供更详细的指引 |
